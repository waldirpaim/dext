# 🛠️ Dext IDE Integration - Roadmap

Este roadmap foca na experiência do desenvolvedor (DX) dentro da IDE do Delphi, trazendo ferramentas modernas de produtividade inspiradas no Visual Studio, Rider e IntelliJ.

> **Visão:** Transformar a IDE do Delphi em um ambiente moderno e produtivo para desenvolvimento Web e Cloud.

---

## 📊 Status Atual: **Beta / Release Candidate 1.0** 🚀

A integração com TestInsight foi concluída, e os componentes de design-time para ORM estão estabilizados.

*Última atualização: 07 de Abril de 2026*

---

## ✅ Funcionalidades Implementadas

### 1. Test Ecosystem Integration
- [x] **TestInsight Support**: Integração total para execução direta de testes Dext na IDE.
- [x] **Auto-Tree Sync**: Resolução de problemas de expansão/seleção automática no TestHost.
- [x] **Runner Status IPC**: Comunicação fluida de progresso entre o console e a IDE.

### 2. Design-Time Experience
- [x] **TEntityDataSet Preview**: Visualização de dados reais em grid no design-time (sem compilar).
- [x] **Dialect Inference**: Detecção automática do dialeto SQL via FireDAC.

---

## 📅 Roadmapped Tools (Próximos Passos)

### 1. Dext Code Inspector (AST-Based)
- [ ] **Smart Navigation**: Indexação e pesquisa rápida (`Ctrl+T`).
- [ ] **Documentation Insight**: Visualização rica de XML Docs em tempo real.

### 2. Scaffolding & Wizards
- [ ] **Project Generator**: Nova UI de "New Project" com seleção de módulos.
- [ ] **Entity-to-Controller Wizard**: Gerador automático de APIs a partir de entidades.

### 3. Integrated Terminal
- [ ] **Dext Terminal Tab**: Aba de terminal dentro da IDE com o Dext CLI integrado.
