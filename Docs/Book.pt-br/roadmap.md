# Roadmap

Conheça o futuro do Framework Dext.

## 🔴 Fase 1: Estabilidade & RC 1.0 (Atual)
Foco em performance do núcleo, completude de recursos padrão e documentação oficial.

- [x] **Versionamento por Path**: Suporte a `/v1/api`.
- [x] **OAuth2 Client Credentials**: Autenticação machine-to-machine empresarial.
- [x] **Rastreamento de Objetos Web**: Gerenciamento robusto de memória no pipeline.
- [x] **Estabilização do Ciclo de Vida DI**: Resolução de escopos e ordem de destruição.
- [x] **Documentação Profissional**: Livro centralizado com paridade EN/PT-BR.

## 🟡 Fase 2: Ecossistema & Experiência do Desenvolvedor
Melhorando o fluxo de trabalho e adicionando utilitários comuns.

- [ ] **Filtros Globais**: Aplicar Action Filters a todos os controllers simultaneamente.
- [ ] **Validação Avançada**: Integração de validação fluente para entidades.
- [ ] **Workers em Segundo Plano (Fila)**: Suporte nativo para processamento assíncrono com retentativas.
- [ ] **Provedores de Logging**: Suporte nativo para ELK, Seq e Sentry.
- [ ] **Evolução do Plugin IDE**: Designers visuais e scaffolding aprimorados.

## 🟢 Fase 3: Futuro & Inovação
Visão de longo prazo para o framework.

- [ ] **Desktop MVU Engine**: Padrão Model-View-Update para Dext.UI.
- [ ] **Parser JSON UTF-8**: Motor JSON de alta performance com zero alocação.
- [ ] **Adaptadores Cloud-Native**: Integração direta com AWS Lambda e Azure Functions.
- [ ] **Evolução de Hubs em Tempo Real**: Escalonamento horizontal para WebSockets.
